# App Icon

## Required

Place a PNG icon file named `icon.png` in this directory before building.

## Specifications

- **Format**: PNG with transparency
- **Size**: 512x512 pixels (minimum)
- **Recommended**: 1024x1024 pixels for best quality

## Platforms

The build process will automatically generate:
- Windows: `.ico` file (multiple sizes)
- Mac: `.icns` file (multiple sizes)

## Where to Get Icons

1. **Design your own** using:
   - Figma
   - Adobe Illustrator
   - Canva

2. **Use icon generators**:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

3. **Purchase from**:
   - https://www.flaticon.com/
   - https://iconscout.com/
